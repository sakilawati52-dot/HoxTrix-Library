## ✨ About

**HoxTrix Library** adalah UI Library modern yang dibuat menggunakan **Luau (Roblox Lua)** untuk mempermudah pembuatan Script Hub dengan tampilan profesional, ringan, dan responsif.

Library ini dirancang agar berjalan dengan baik pada berbagai executor Roblox baik **Mobile** maupun **PC**, serta memiliki desain minimalis dengan performa tinggi.

---

## ⭐ Features

- 🎨 Modern Dark UI
- 📱 Mobile & PC Friendly
- 🛡️ Safe CoreGui / gethui Support
- 🖱️ Smooth Draggable Window
- 📂 Dynamic Sidebar Tabs
- ⚡ Fast & Lightweight
- 📜 Automatic Scrolling
- 🔘 Buttons
- ✅ Toggles
- 🚀 Easy API
- 💙 Clean Animations
